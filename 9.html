F

рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin\left(\frac{2x}{3}\right) + \tan\left(\frac{4x}{5}\right)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\sin\left(\frac{2x}{3}\right)$ бһӮбһә $p_1 = \frac{2\pi}{2/3} = 3\pi$
- бһҒбһҪбһ”бһ“бҹғ $\tan\left(\frac{4x}{5}\right)$ бһӮбһә $p_2 = \frac{\pi}{4/5} = \frac{5\pi}{4}$
- бһҒбһҪбһ”бһҹбһҡбһ»бһ” $p = BCNN\left(3\pi, \frac{5\pi}{4}\right) = \frac{BCNN(3, 5)}{PGCD(1, 4)}\pi = \frac{15}{1}\pi = 15\pi$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+1) = \frac{f(x)-1}{f(x)+1}$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- $f(x+2) = \frac{f(x+1)-1}{f(x+1)+1} = \frac{\frac{f(x)-1}{f(x)+1}-1}{\frac{f(x)-1}{f(x)+1}+1} = \frac{-2}{2f(x)} = -\frac{1}{f(x)}$
- $f(x+4) = -\frac{1}{f(x+2)} = -\frac{1}{-1/f(x)} = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = 4$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = (-1)^{[x]}$ (бһҠбҹӮбһӣ $[x]$ бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһ“бҹғ $x$)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f(x+2) = (-1)^{[x+2]} = (-1)^{[x]+2} = (-1)^{[x]} \cdot (-1)^2 = (-1)^{[x]} = f(x)$бҹ”
бһ”бһҫ $p=1$ бһ“бҹ„бҹҮ $f(x+1) = (-1)^{[x]+1} = -f(x) \neq f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = 2$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(\sqrt{2}x) + \cos(\sqrt{3}x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғбһҸбһҪбһ‘бһёбҹЎ бһӮбһә $p_1 = \frac{2\pi}{\sqrt{2}} = \sqrt{2}\pi$ бһ“бһ·бһ„бһҸбһҪбһ‘бһёбҹў бһӮбһә $p_2 = \frac{2\pi}{\sqrt{3}} = \frac{2\sqrt{3}\pi}{3}$бҹ”
- бһ•бһӣбһ’бҹҖбһ”бһҒбһҪбһ” $\frac{p_1}{p_2} = \frac{\sqrt{2}\pi}{\frac{2\sqrt{3}\pi}{3}} = \frac{3\sqrt{2}}{2\sqrt{3}} = \frac{\sqrt{6}}{2}$ бһҠбҹӮбһӣбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһўбһҹбһ“бһ·бһ‘бһ¶бһ“бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ•бһӣбһ”бһјбһҖбһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбһ¶бһ“бһҒбһҪбһ”бһЎбһҫбһҷбҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+2) + f(x) = f(x+1)$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбҹ– $f(x+2) = f(x+1) - f(x)$
- $f(x+3) = f(x+2) - f(x+1) = (f(x+1) - f(x)) - f(x+1) = -f(x)$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(x+6) = -f(x+3) = -(-f(x)) = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = 6$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \tan(\pi x) - [\tan(\pi x)]$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҖбһ“бҹ’бһҹбҹ„бһҳбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ”бҹ’бһҡбһ—бһ¶бһӮ $f(x) = \{ \tan(\pi x) \} $бҹ” бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\tan(\pi x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = \frac{\pi}{\pi} = 1$бҹ” бһҖбһ¶бһҡбһ”бҹҶбһ–бһ¶бһҖбҹӢбһ•бҹ’бһ“бҹӮбһҖбһ”бҹ’бһҡбһ—бһ¶бһӮбһӣбһҫбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ” бһҳбһ·бһ“бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһңбҹӮбһ„бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһҡбһ”бһҹбҹӢбһңбһ¶бһЎбһҫбһҷ бһӮбһә $p = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣ $I = \int_{0}^{2026\pi} |\sin x| \, dx$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = |\sin x|$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = \pi$бҹ”
бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбһҪбһ”бҹ– $I = 2026 \times \int_{0}^{\pi} \sin x \, dx = 2026 \times [-\cos x]_{0}^{\pi} = 2026 \times (1 - (-1)) = 2026 \times 2 = 4052$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+3) = \frac{1+f(x)}{1-f(x)}$бҹ” бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғ $f(2030)$ бһ”бһҫбһҠбһ№бһ„бһҗбһ¶ $f(2) = 5$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҳбһёбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһңбһ¶бһҹбҹ’бһҳбһҫбһ“бһ№бһ„ $4 \times 3 = 12$ бһҮбҹҶбһ бһ¶бһ“ (бһҠбһјбһ…бһӣбҹҶбһ бһ¶бһҸбҹӢбһҳбһ»бһ“бҹ—) бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(x+12) = f(x)$бҹ”
- бһҷбһҫбһ„бһҳбһ¶бһ“ $2030 = 12 \times 169 + 2$ бһҠбһјбһ…бһ“бҹҒбҹҮ $f(2030) = f(2) = 5$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \cos(\sin x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f(x+\pi) = \cos(\sin(x+\pi)) = \cos(-\sin x) = \cos(\sin x) = f(x)$ (бһ–бҹ’бһҡбҹ„бҹҮ $\cos(-t) = \cos t$)бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = \pi$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(\cos x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһ”бһҫбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $p=\pi$бҹ– $f(x+\pi) = \sin(\cos(x+\pi)) = \sin(-\cos x) = -\sin(\cos x) \neq f(x)$бҹ”
- бһ”бһҫбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $p=2\pi$бҹ– $f(x+2\pi) = \sin(\cos(x+2\pi)) = \sin(\cos x) = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $2\pi$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+1) + f(x-1) = \sqrt{3}f(x)$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲ Recurrence Relation бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҳбһ»бҹҶбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡ $\cos\left(\frac{\pi}{6}\right) = \frac{\sqrt{3}}{2}$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһңбһ¶бһҹбҹ’бһҳбһҫбһ“бһ№бһ„ $12 \times \text{бһҮбҹҶбһ бһ¶бһ“} = 12 \times 1 = 12$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \cos^8 x - \sin^8 x$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбһ¶бһҖбҹӢбһҮбһ¶бһ•бһӣбһӮбһ»бһҺбҹ– $f(x) = (\cos^4 x - \sin^4 x)(\cos^4 x + \sin^4 x) = (\cos^2 x - \sin^2 x)(\cos^2 x + \sin^2 x)(1 - \frac{1}{2}\sin^2(2x)) = \cos(2x)(1 - \frac{1}{2}\sin^2(2x))$бҹ”
бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f(x+\pi) = \cos(2x+2\pi)(1 - \frac{1}{2}\sin^2(2x+2\pi)) = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $\pi$бҹ”
бһҸбһҫбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = x - [x] + \cos(\pi x)$ бһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҸбһҪбһ‘бһёбһҳбһҪбһҷ $x - [x] = \{x\}$ бһҳбһ¶бһ“бһҒбһҪбһ” $p_1 = 1$
- бһҸбһҪбһ‘бһёбһ–бһёбһҡ $\cos(\pi x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $p_2 = \frac{2\pi}{\pi} = 2$
- бһҒбһҪбһ”бһҹбһҡбһ»бһ”бһӮбһә $p = BCNN(1, 2) = 2$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+4) = \sqrt{f(x) - f^2(x)} + \frac{1}{2}$ бһ…бҹҶбһ–бҹ„бҹҮ $\frac{1}{2} \le f(x) \le 1$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһӣбһҫбһҖбһҮбһ¶бһҖбһ¶бһҡбҹҒбҹ– $\left(f(x+4) - \frac{1}{2}\right)^2 = f(x) - f^2(x) = \frac{1}{4} - \left(f(x) - \frac{1}{2}\right)^2$бҹ”
бһҸбһ¶бһ„ $g(x) = \left(f(x) - \frac{1}{2}\right)^2 \Rightarrow g(x+4) = \frac{1}{4} - g(x)$бҹ”
бһ“бһ¶бҹҶбһұбҹ’бһҷ $g(x+8) = \frac{1}{4} - g(x+4) = g(x) \Rightarrow f(x+8) = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = 8$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \ln(|\sin x| + |\cos x|)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f\left(x+\frac{\pi}{2}\right) = \ln\left(\left|\sin\left(x+\frac{\pi}{2}\right)\right| + \left|\cos\left(x+\frac{\pi}{2}\right)\right|\right) = \ln(|\cos x| + |-\sin x|) = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = \frac{\pi}{2}$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = e^{\cos(2x)} \cdot \sqrt{\tan^2 x + 1}$ бһӣбһҫбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- $\sqrt{\tan^2 x + 1} = \sqrt{\sec^2 x} = |\sec x|$ бһҳбһ¶бһ“бһҒбһҪбһ” $\pi$бҹ”
- $e^{\cos(2x)}$ бһҖбҹҸбһҳбһ¶бһ“бһҒбһҪбһ” $\pi$ бһҠбҹӮбһҡбҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бһӣбһӮбһ»бһҺбһҡбһ”бһҹбҹӢбһңбһ¶бһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = \pi$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+p) = f(x)$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\int_{a}^{a+2p} f(x) \, dx$ бһ”бһҫбһҠбһ№бһ„бһҗбһ¶ $\int_{0}^{p} f(x) \, dx = 7$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһӮбҹ’бһҡбһ№бҹҮбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ” бһҸбһҳбҹ’бһӣбҹғбһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣбһӣбһҫбһ…бһ“бҹ’бһӣбҹ„бҹҮбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһңбҹӮбһ„бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ–бһ бһ»бһӮбһ»бһҺбһ“бҹғбһҒбһҪбһ” ($2p$) бһӮбһәбһҳбһ·бһ“бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ“бһ№бһ„бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ $a$ бһЎбһҫбһҷбҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһңбһ¶бһҹбҹ’бһҳбһҫбһ“бһ№бһ„ $2 \times \int_{0}^{p} f(x) \, dx = 2 \times 7 = 14$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(x) + \left\{ \frac{x}{2\pi} \right\}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\sin(x)$ бһӮбһә $p_1 = 2\pi$
- бһҒбһҪбһ”бһ“бҹғ $\left\{ \frac{x}{2\pi} \right\}$ бһӮбһә $p_2 = \frac{1}{1/(2\pi)} = 2\pi$
- бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҹбһҡбһ»бһ”бһӮбһә $p = BCNN(2\pi, 2\pi) = 2\pi$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+1) = 1 - f(x)$бҹ” бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ•бһӣбһ”бһјбһҖ $S = f(1) + f(2) + f(3) + \dots + f(100)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһғбһҫбһүбһҗбһ¶ $f(x) + f(x+1) = 1$бҹ”
бһҠбһјбһ…бһ“бҹҒбҹҮбҹ– $f(1) + f(2) = 1$, $f(3) + f(4) = 1$, $\dots$, $f(99) + f(100) = 1$бҹ”
бһ•бһӣбһ”бһјбһҖбһҳбһ¶бһ“бһҖбһјбһ“бһӮбһјбһҹбһҡбһ»бһ”бһ…бҹҶбһ“бһҪбһ“ $50$ бһӮбһј бһ“бһ¶бҹҶбһұбҹ’бһҷ $S = 50 \times 1 = 50$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \frac{\sin(3x)}{\sin x} - \frac{\cos(3x)}{\cos x}$ бһӣбһҫбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһҸбһҳбҹ’бһҡбһјбһңбһ—бһ¶бһӮбһ”бҹӮбһ„бҹ– $f(x) = \frac{\sin(3x)\cos x - \cos(3x)\sin x}{\sin x \cos x} = \frac{\sin(3x-x)}{\frac{1}{2}\sin(2x)} = \frac{\sin(2x)}{\frac{1}{2}\sin(2x)} = 2$бҹ”
бһҠбҹ„бһҷбһҹбһ¶бһҡбһңбһ¶бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҗбҹҒбһҡ бһңбһ¶бһҳбһ·бһ“бһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһЎбһҫбһҷбҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = e^{\ln(\cos^2 x)}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбҹ„бһҖбһ¶бһҡбһёбһҸбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ¶бһ“ $f(x) = \cos^2 x$ (бһ…бҹҶбһ–бҹ„бҹҮ $\cos x \neq 0$)бҹ”
бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\cos^2 x = \frac{1+\cos(2x)}{2}$ бһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = \pi$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+5) \cdot f(x) + f(x+5) + f(x) = 0$ бһ…бҹҶбһ–бҹ„бҹҮ $f(x) \neq -1$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ‘бһ¶бһүбһҡбһҖ $f(x+5) = \frac{-f(x)}{f(x)+1}$бҹ”
бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(x+10) = \frac{-f(x+5)}{f(x+5)+1} = \frac{-\frac{-f(x)}{f(x)+1}}{\frac{-f(x)}{f(x)+1}+1} = \frac{f(x)}{1} = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = 10$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \cos(2\pi x) \cdot \{x\}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\cos(2\pi x)$ бһӮбһә $p_1 = \frac{2\pi}{2\pi} = 1$
- бһҒбһҪбһ”бһ“бҹғ $\{x\}$ бһӮбһә $p_2 = 1$
- бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бһӣбһӮбһ»бһҺбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбһҡбһҖбҹ’бһҹбһ¶бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһҡбһҪбһҳбһӮбһә $p = 1$бҹ”
бһҸбһҫбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(x) + \cos(\pi x)$ бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ”бһҠбҹӮбһҡбһ¬бһ‘бҹҒ? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\sin x$ бһӮбһә $2\pi$ (бһ…бҹҶбһ“бһҪбһ“бһўбһҹбһ“бһ·бһ‘бһ¶бһ“)бҹ”
- бһҒбһҪбһ”бһ“бҹғ $\cos(\pi x)$ бһӮбһә $\frac{2\pi}{\pi} = 2$ (бһ…бҹҶбһ“бһҪбһ“бһҹбһ“бһ·бһ‘бһ¶бһ“)бҹ”
- бһ•бһӣбһ’бҹҖбһ”бһҡбһңбһ¶бһ„бһҒбһҪбһ”бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ $\frac{2\pi}{2} = \pi$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһўбһҹбһ“бһ·бһ‘бһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷбһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһҡбһҖбһ”бһ–бҹ’бһңбһҮбһ¶бһ–бһ бһ»бһӮбһ»бһҺбһҡбһҪбһҳбһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһ”бһ¶бһ“бһЎбһҫбһҷбҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһңбһ¶бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ”бһ‘бҹҒбҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \frac{1}{2-\cos(3x)}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ”бҹ’бһҡбһ—бһ¶бһӮбһ“бҹҒбҹҮбһӮбһәбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһ‘бҹ…бһӣбһҫбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбһ“бҹ…бһ—бһ¶бһӮбһ”бҹӮбһ„бҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡ $\cos(3x)$ бһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p = \frac{2\pi}{3}$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ…бһҳбҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶бһҖбҹҸбһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p = \frac{2\pi}{3}$ бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”
