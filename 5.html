рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin\left(\frac{2x}{3}\right) + \cos\left(\frac{3x}{4}\right)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\sin\left(\frac{2x}{3}\right)$ бһӮбһә $p_1 = \frac{2\pi}{2/3} = 3\pi$
- бһҒбһҪбһ”бһ“бҹғ $\cos\left(\frac{3x}{4}\right)$ бһӮбһә $p_2 = \frac{2\pi}{3/4} = \frac{8\pi}{3}$
- бһҒбһҪбһ”бһҹбһҡбһ»бһ” $p = BCNN\left(3\pi, \frac{8\pi}{3}\right) = BCNN\left(\frac{9\pi}{3}, \frac{8\pi}{3}\right) = \frac{BCNN(9,8)\pi}{3} = \frac{72\pi}{3} = 24\pi$бҹ”
бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = x - [x]$ бһҠбҹӮбһӣ $[x]$ бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһ“бҹғ $x$бҹ” бһҸбһҫ $f$ бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ”бҹ’бһҡбһ—бһ¶бһӮ $f(x) = \{x\} = x - [x]$бҹ” бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһ“бҹғбһ•бҹ’бһ“бҹӮбһҖбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢ $[x+1] = [x] + 1$бҹ”
бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(x+1) = (x+1) - [x+1] = x + 1 - ([x] + 1) = x - [x] = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = 1$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+2) + f(x) = 0$бҹ” бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $f(2026)$ бһ”бһҫбһҠбһ№бһ„бһҗбһ¶ $f(2) = 5$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡ $f(x+2) = -f(x)$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(x+4) = -f(x+2) = -(-f(x)) = f(x)$ бһҠбһјбһ…бһ“бҹҒбҹҮ $f$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = 4$бҹ”
бһҠбҹ„бһҷбһҹбһ¶бһҡ $2026 = 2 + 2024 = 2 + 506 \times 4$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(2026) = f(2) = 5$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \tan(3x) \cdot \cot(3x)$ бһӣбһҫбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶ $f(x) = 1$ бһӣбһҫбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҸбһҠбҹ„бһҷ бһҖбҹҸбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ“бһ№бһ„ $\tan(3x)$ бһ“бһ·бһ„ $\cot(3x)$ бһҠбҹӮбһҡбҹ”
бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһӮбһә $3x \neq \frac{k\pi}{2} \Rightarrow x \neq \frac{k\pi}{6}$бҹ” бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҠбһ¶бһ…бҹӢбҹ—бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ $1$ бһ бһҫбһҷбһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһҹбһ—бһ¶бһ–бһҠбһҫбһҳбһҡбһ¶бһӣбҹӢбһ”бҹ’бһҡбһңбҹӮбһ„ $p = \frac{\pi}{6}$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \cos(\sin x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f(x+\pi) = \cos(\sin(x+\pi)) = \cos(-\sin x) = \cos(\sin x) = f(x)$ (бһ–бҹ’бһҡбҹ„бҹҮбһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһӮбһј)бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = \pi$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin^4 x + \cos^4 x$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бҹ– $f(x) = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x\cos^2 x = 1 - \frac{1}{2}(2\sin x\cos x)^2 = 1 - \frac{1}{2}\sin^2(2x) = 1 - \frac{1}{2}\left(\frac{1-\cos(4x)}{2}\right)$бҹ” бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\cos(4x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = \frac{2\pi}{4} = \frac{\pi}{2}$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$, $f(x+3) = \frac{f(x)-5}{f(x)+1}$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- $f(x+6) = \frac{f(x+3)-5}{f(x+3)+1} = \frac{\frac{f(x)-5}{f(x)+1}-5}{\frac{f(x)-5}{f(x)+1}+1} = \frac{-4f(x)-10}{2f(x)-4} = \frac{2f(x)+5}{2-f(x)}$
- $f(x+12) = \frac{2f(x+6)+5}{2-f(x+6)} = \dots = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = 4 \times 3 = 12$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(\sqrt{3}x) + \cos(x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\sin(\sqrt{3}x)$ бһӮбһә $p_1 = \frac{2\pi}{\sqrt{3}}$ бһ бһҫбһҷбһҒбһҪбһ”бһ“бҹғ $\cos(x)$ бһӮбһә $p_2 = 2\pi$бҹ”
- бһ•бһӣбһ’бҹҖбһ” $\frac{p_1}{p_2} = \frac{2\pi/\sqrt{3}}{2\pi} = \frac{1}{\sqrt{3}}$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһўбһҹбһ“бһ·бһ‘бһ¶бһ“бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бһӣбһ”бһјбһҖбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ”бһЎбһҫбһҷбҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = (-1)^{[x]}$ бһҠбҹӮбһӣ $[x]$ бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһ“бҹғ $x$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f(x+2) = (-1)^{[x+2]} = (-1)^{[x]+2} = (-1)^{[x]} \cdot (-1)^2 = (-1)^{[x]} = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = 2$ (бһ–бҹ’бһҡбҹ„бҹҮбһ”бһҫ $p=1$, $(-1)^{[x+1]} = -(-1)^{[x]} \neq f(x)$)бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \frac{\sin(x) + \sin(3x)}{\cos(x) + \cos(3x)}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ•бһӣбһ”бһјбһҖбһ‘бҹ…бһҮбһ¶бһ•бһӣбһӮбһ»бһҺбҹ–
$f(x) = \frac{2\sin(2x)\cos(x)}{2\cos(2x)\cos(x)} = \tan(2x)$бҹ” бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\tan(2x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = \frac{\pi}{2}$бҹ”
бһ”бһҫбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = 5$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ $\int_{0}^{15} f(x) \, dx$ бһ”бһҫбһӮбҹҒбһҠбһ№бһ„бһҗбһ¶ $\int_{0}^{5} f(x) \, dx = 4$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ”бҹ– $\int_{0}^{np} f(x) \, dx = n \int_{0}^{p} f(x) \, dx$бҹ”
бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ $p = 5, n = 3$ (бһ–бҹ’бһҡбҹ„бҹҮ $15 = 3 \times 5$) бһ“бһ¶бҹҶбһұбҹ’бһҷбһҹбһҡбһ»бһ”бһӮбһә $3 \times 4 = 12$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin^6 x + \cos^6 x$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһҸбһ¶бһҳбһҜбһҖбһӣбһҖбҹ’бһҒбһҺбһ—бһ¶бһ–бҹ– $f(x) = 1 - 3\sin^2 x\cos^2 x = 1 - \frac{3}{4}\sin^2(2x) = 1 - \frac{3}{4}\left(\frac{1-\cos(4x)}{2}\right)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһәбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ“бһ№бһ„ $\cos(4x)$ бһҠбҹӮбһӣбһҳбһ¶бһ“ $p = \frac{2\pi}{4} = \frac{\pi}{2}$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+1) + f(x+3) = f(x)$бҹ” бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ $f$ бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҒбһҪбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһҒбһҪбһ” $p$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҮбҹҶбһ“бһҪбһҹ $x \to x+1 \Rightarrow f(x+2) + f(x+4) = f(x+1)$
бһҮбҹҶбһ“бһҪбһҹ $x \to x+3 \Rightarrow f(x+4) + f(x+6) = f(x+3)$
бһ”бһјбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбҹ– $f(x+2) + 2f(x+4) + f(x+6) = f(x+1)+f(x+3) = f(x)$бҹ” бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһҷбһҫбһ„бһ“бһ№бһ„бһ‘бһ¶бһүбһ”бһ¶бһ“ $f(x+8) = f(x)$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҒбһҪбһ” $p = 8$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = |\cos x| + |\sin x|$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f\left(x+\frac{\pi}{2}\right) = \left|\cos\left(x+\frac{\pi}{2}\right)\right| + \left|\sin\left(x+\frac{\pi}{2}\right)\right| = |-\sin x| + |\cos x| = |\sin x| + |\cos x| = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $p = \frac{\pi}{2}$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(\pi x) - [ \pi x ]$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\sin(\pi x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $p_1 = \frac{2\pi}{\pi} = 2$
- бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\pi x - [\pi x] = \{\pi x\}$ бһҳбһ¶бһ“бһҒбһҪбһ” $p_2 = \frac{1}{\pi}$
- бһҖбҹҸбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһҫбһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҪбһҹбһҡбһ»бһ” $f(x+2) = \sin(\pi x + 2\pi) + \{\pi x + 2\pi\} = \sin(\pi x) + \{\pi x\} = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҹбһҡбһ»бһ”бһӮбһә $p = 2$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sqrt{\sin^2(3x)}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶ $f(x) = \sqrt{\sin^2(3x)} = |\sin(3x)|$бҹ” бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\sin(3x)$ бһҳбһ¶бһ“бһҒбһҪбһ” $\frac{2\pi}{3}$ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҠбҹ„бһҷбһҹбһ¶бһҡбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸ бһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһңбһ¶бһҸбҹ’бһҡбһјбһңбһҗбһҷбһ…бһ»бҹҮбһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣ бһӮбһә $p = \frac{\pi}{3}$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$, $f(x+p) = f(x)$ бһ“бҹ„бҹҮбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $g(x) = f(ax+b)$ бһҳбһ¶бһ“бһҒбһҪбһ”бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ¶бһ“ $g(x+p') = g(x) \Rightarrow f(a(x+p')+b) = f(ax+b) \Rightarrow f(ax+b + ap') = f(ax+b)$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷ $ap' = p \Rightarrow p' = \frac{p}{a}$ бһ¬бһҮбһ¶бһ‘бһјбһ‘бҹ…бһӮбһә $\frac{p}{|a|}$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin(x) + \cos\left(\frac{x}{2}\right) + \sin\left(\frac{x}{3}\right)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҒбһҪбһ”бһ“бҹғ $\sin(x)$ бһӮбһә $2\pi$
- бһҒбһҪбһ”бһ“бҹғ $\cos(x/2)$ бһӮбһә $4\pi$
- бһҒбһҪбһ”бһ“бҹғ $\sin(x/3)$ бһӮбһә $6\pi$
- бһҒбһҪбһ”бһҹбһҡбһ»бһ”бһӮбһә $BCNN(2\pi, 4\pi, 6\pi) = 12\pi$бҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+1) = 1 - f(x)$бҹ” бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $f(100)$ бһ”бһҫбһҠбһ№бһ„бһҗбһ¶ $f(0) = 2$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһҡбһҖбһҒбһҪбһ”бҹ– $f(x+2) = 1 - f(x+1) = 1 - (1 - f(x)) = f(x)$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $f$ бһҳбһ¶бһ“бһҒбһҪбһ” $p = 2$бҹ”
бһҠбҹ„бһҷбһҹбһ¶бһҡ $100 = 50 \times 2$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $f(100) = f(0) = 2$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \cos(\sqrt{2}x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҳбҹҒбһӮбһ»бһҺ $a = \sqrt{2}$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһўбһҹбһ“бһ·бһ‘бһ¶бһ“бһҖбҹҸбһҠбҹ„бһҷ бһҖбҹҸбһңбһ¶бһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ‘бҹ„бһӣ бһҠбһјбһ…бһ“бҹҒбҹҮбһңбһ¶бһ“бҹ…бһҸбҹӮбһҳбһ¶бһ“бһҒбһҪбһ”бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $p = \frac{2\pi}{|a|} = \frac{2\pi}{\sqrt{2}} = \sqrt{2}\pi$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = 3\sin(2x) + 4\cos(2x)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹӮбһҳбһҪбһҷбҹ– $f(x) = 5\sin(2x + \alpha)$ бһҠбҹӮбһӣ $\tan\alpha = \frac{4}{3}$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһңбһ¶бһӮбһә $p = \frac{2\pi}{2} = \frac{\pi}{2}$ бһҳбһ·бһ“бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҳбҹҒбһӮбһ»бһҺбһ”бһ“бҹ’бһҹбҹҶбһЎбһҫбһҷбҹ”
бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢ $x \in \mathbb{R}$ бһӮбҹҒбһҳбһ¶бһ“ $f(x+3) + f(x-3) = \sqrt{2}f(x)$бҹ” бһҡбһҖбһҒбһҪбһ” $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһ„ $t = x-3 \Rightarrow f(t+6) + f(t) = \sqrt{2}f(t+3) \Rightarrow f(t+6) = \sqrt{2}f(t+3) - f(t)$бҹ”
бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҹбһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһҷбһҫбһ„бһ“бһ№бһ„бһ‘бһ¶бһүбһ”бһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $f(t+12) = -f(t) \Rightarrow f(t+24) = f(t)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһӮбһә $p = 24$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \frac{1}{2-\cos(5x)}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ”бҹ’бһҡбһ—бһ¶бһӮбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҒбһҪбһ”бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһҸбҹӮбһ‘бҹ…бһӣбһҫбһҸбһҪбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡ $\cos(5x)$ бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһңбһ¶бһӮбһә $p = \frac{2\pi}{5}$бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \sin^3 x$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ–бҹ’бһҡбҹҮбһӣбһёбһ“бҹҒбһўбҹҠбҹӮбһҡбҹ– $\sin^3 x = \frac{3\sin x - \sin(3x)}{4}$бҹ”
- бһҒбһҪбһ”бһ“бҹғ $\sin x$ бһӮбһә $2\pi$ бһ“бһ·бһ„бһҒбһҪбһ”бһ“бҹғ $\sin(3x)$ бһӮбһә $\frac{2\pi}{3}$бҹ”
- бһҒбһҪбһ”бһҹбһҡбһ»бһ”бһӮбһә $BCNN\left(2\pi, \frac{2\pi}{3}\right) = 2\pi$ (бһ–бҹ’бһҡбҹ„бҹҮбһҹбһёбһ“бһ»бһҹбһҹбҹ’бһңбҹҗбһҷбһӮбһ»бһҺбһҹбҹҒбһҹбһҡбһҖбҹ’бһҹбһ¶бһҒбһҪбһ” $2\pi$ бһҠбһҠбҹӮбһӣ)бҹ”
бһҡбһҖбһҒбһҪбһ”бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸ $p$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \ln(\tan x)$ бһӣбһҫбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶бһ…бҹҶбһ–бҹ„бҹҮ $x \in \left(0, \frac{\pi}{2}\right)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҷбһҫбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ $f(x+\pi) = \ln(\tan(x+\pi)) = \ln(\tan x) = f(x)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһңбһ¶бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһӣбҹ„бһҖбһ¶бһҡбһёбһҸ бһҖбҹҸбһңбһ¶бһ“бҹ…бһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶бһҒбһҪбһ”бһҡбһ”бһҹбҹӢбһҸбһ„бҹӢбһҹбһ„бҹӢбһҠбһҠбҹӮбһӣ бһӮбһә $p = \pi$бҹ”
