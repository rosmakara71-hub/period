២. វិធីដោះស្រាយសមីការអនុគមន៍រកខួប (Functional Equations)

ប្រើវិធីជំនួសតួ x → x + a ដើម្បីបង្កើតទំនាក់ទំនងវិលជុំ៖

  • បើ f(x + a) = -f(x) នោះខួបគឺ p = 2a
  • បើ f(x + a) = 1 / [1 - f(x)] នោះវាវិលជុំ ៣ ជំហាន នាំឱ្យខួបគឺ p = 3a
  • បើ f(x + a) = [1 - f(x)] / [1 + f(x)] នោះវាវិលជុំ ២ ជំហាន នាំឱ្យខួបគឺ p = 2a
  • បើ f(x + a) = [f(x) - 1] / [f(x) + 1] នោះវាវិលជុំ ៤ ជំហាន នាំឱ្យខួបគឺ p = 4a

៣. លក្ខណៈខួបជាមួយផ្នែកចំនួនគត់ [x] និងផ្នែកប្រភាគ {x}

  • ផ្នែកប្រភាគ {kx} = x - [x] មានខួបថេរ p = 1 / |k| ជានិច្ច។
  • អនុគមន៍ទម្រង់ (-1)[x] ឬ cos(π[x]) ប្តូរសញ្ញារាល់ពេល x កើនឡើង 1 ឯកតា នាំឱ្យខួបតូចបំផុតរបស់វាគឺ p = 2។
