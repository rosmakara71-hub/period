ការស្វែងរកខួបនៃអនុគមន៍ (បែងចែកតាមកម្រិត)

កម្រិតទី ១ ៖ មូលដ្ឋានគ្រឹះ (Basic) ថ្នាក់ទី ១១-១២

និយមន័យ៖ រកចំនួនពិតវិជ្ជមាន \(T\) តូចបំផុតដែលផ្ទៀងផ្ទាត់ \(f(x+T) = f(x)\)។

រូបមន្តសម្រាប់អនុគមន៍ទោលគ្រឹះ \( (b \neq 0) \):

អនុគមន៍ \(\sin(bx)\) និង \(\cos(bx)\) \(\implies T = \frac{2\pi}{|b|}\)

អនុគមន៍ \(\tan(bx)\) និង \(\cot(bx)\) \(\implies T = \frac{\pi}{|b|}\)
ឧទាហរណ៍៖ រកខួបនៃ \(f(x) = \cos(4x)\)
ដំណោះស្រាយ៖ តាមរូបមន្តយើងមាន \(b = 4\) ដូចនេះ ខួបគឺ \(T = \frac{2\pi}{4} = \frac{\pi}{2}\)។
កម្រិតទី ២ ៖ កម្រិតមធ្យម (Intermediate) ត្រៀមប្រឡងបាក់ឌុប

១. ករណីមានស្វ័យគុណគូ ឬតម្លៃដាច់ខាត

នៅពេលអនុគមន៍ស៊ីនុស ឬកូស៊ីនុសមានស្វ័យគុណជាចំនួនគូ ឬនៅក្នុងតម្លៃដាច់ខាត ខួបរបស់វានឹងថយមកនៅត្រឹម \(\frac{\pi}{|b|}\)៖

\(f(x) = \sin^2(bx)\) ឬ \(f(x) = |\cos(bx)| \implies T = \frac{\pi}{|b|}\)

២. ករណីផលបូក/ផលដកអនុគមន៍ \(f(x) \pm g(x)\)

ខួបរួមគឺជានិរន្តរគូករួមតូចបំផុត (LCM) នៃខួបទាំងពីរ៖ \(T = \text{LCM}(T_1, T_2)\)

ឧទាហរណ៍៖ រកខួបនៃ \(F(x) = \sin(2x) + \cos(3x)\)
- ខួបនៃ \(\sin(2x)\) គឺ \(T_1 = \frac{2\pi}{2} = \pi\)
- ខួបនៃ \(\cos(3x)\) គឺ \(T_2 = \frac{2\pi}{3}\)
- នាំឲ្យខួបរួម \(T = \text{LCM}(\pi, \frac{2\pi}{3}) = 2\pi\)។
កម្រិតទី ៣ ៖ កម្រិតខ្ពស់ (Advanced) ប្រឡងសិស្សពូកែ / និស្សិតសាកលវិទ្យាល័យ

១. លក្ខខណ្ឌអត្ថិភាពនៃខួប (Existence of Period)

ផលបូកនៃអនុគមន៍ខួបពីរ មានខួបរួមលុះត្រាតែផលធៀប \(\frac{T_1}{T_2}\) ជា ចំនួនសនិទាន (\(\mathbb{Q}\))។

*សំគាល់៖ \(\sin(x) + \sin(\pi x)\) គ្មានខួបទេ ព្រោះ \(\frac{2\pi}{2} = \pi \notin \mathbb{Q}\)។

២. ករណីពិសេសបំពេញគ្នា (Symmetric functions)

ចំពោះអនុគមន៍ខ្លះ ថ្វីត្បិតតែតាមរូបមន្តរកឃើញ LCM ធំ ប៉ុន្តែដោយសារលក្ខណៈឆ្លុះ វានាំឲ្យមានខួបតូចជាង LCM នោះទៅទៀត។

ឧទាហរណ៍ពិសេស៖ \(f(x) = |\sin x| + |\cos x|\)
តាមរូបមន្តកម្រិត២៖ \(T_1 = \pi, T_2 = \pi \implies \text{LCM} = \pi\)។
ប៉ុន្តែបើផ្ទៀងផ្ទាត់តាមនិយមន័យជាមួយ \(T = \frac{\pi}{2}\)៖
\(f(x + \frac{\pi}{2}) = |\sin(x + \frac{\pi}{2})| + |\cos(x + \frac{\pi}{2})| = |\cos x| + |- \sin x| = f(x)\)។
ដូចនេះ ខួបពិតប្រាកដតូចបំផុតគឺ \(T = \frac{\pi}{2}\)។
