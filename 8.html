បំប្លែង៖ $f(x) = \frac{\sin x}{\cos x} - \frac{\cos x}{\sin x} = \frac{\sin^2 x - \cos^2 x}{\sin x \cos x} = \frac{-\cos(2x)}{\frac{1}{2}\sin(2x)} = -2\cot(2x)$។
អនុគមន៍កូតង់សង់ $\cot(2x)$ មានខួបតូចបំផុត $p = \frac{\pi}{2}$។
*(សម្គាល់៖ ទោះបីជាបំប្លែងបាន $\cot(2x)$ ដែលមានខួប $\frac{\pi}{2}$ ប៉ុន្តែដោយសារមានការដកតួ $f(x+\frac{\pi}{2}) = \tan(x+\pi/2) - \cot(x+\pi/2) = -\cot x + \tan x = f(x)$ នាំឱ្យខួបពិតប្រាកដគឺ $\frac{\pi}{2}$)*។
បើគ្រប់ $x \in \mathbb{R}$ គេមាន $f(x+a) = -f(x)$។ បង្ហាញថា $f$ ជាអនុគមន៍ខួបដែលមានខួប $p$ ស្មើនឹង៖ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
យើងជំនួស $x$ ដោយ $x+a$ ក្នុងសមីការដើម៖ $f(x+2a) = -f(x+a) = -(-f(x)) = f(x)$។ ដូចនេះតាមនិយមន័យនាំឱ្យខួបគឺ $p = 2a$។
រកខួបតូចបំផុត $p$ នៃអនុគមន៍ $f(x) = \cos(2x)\cos(4x)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
បំប្លែងផលគុណទៅជាផលបូក៖ $f(x) = \frac{1}{2}[\cos(6x) + \cos(2x)]$ biographies។
- ខួបនៃ $\cos(6x)$ គឺ $p_1 = \frac{2\pi}{6} = \frac{\pi}{3}$
- ខួបនៃ $\cos(2x)$ គឺ $p_2 = \frac{2\pi}{2} = \pi$
- ខួបសរុបគឺ $BCNN\left(\frac{\pi}{3}, \pi\right) = \pi$។
រកខួបតូចបំផុត $p$ នៃអនុគមន៍ $f(x) = \sin^4 x$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ពង្រួមដឺក្រេជាជំហានៗ៖ $f(x) = \left(\frac{1-\cos(2x)}{2}\right)^2 = \frac{1 - 2\cos(2x) + \cos^2(2x)}{4} = \frac{1 - 2\cos(2x) + \frac{1+\cos(4x)}{2}}{4}$។
តួដែលមានខួបធំជាងគេក្នុងកន្សោមគឺ $\cos(2x)$ ដែលមានខួប $p = \frac{2\pi}{2} = \pi$ (ខណៈពេល $\cos(4x)$ មានខួប $\frac{\pi}{2}$)។ ដូចនេះខួបសរុបគឺ $BCNN\left(\pi, \frac{\pi}{2}\right) = \pi$។
